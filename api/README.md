## 📦 JSON Server API

### 👤 **Users**

| Method | Endpoint     | Description  |
| ------ | ------------ | ------------ |
| POST   | `/users`     | Sign in      |
| GET    | `/users/:id` | Get one user |

### 🥤 **Machines**

| Method | Endpoint        | Description      |
| ------ | --------------- | ---------------- |
| GET    | `/machines`     | Get all machines |
| GET    | `/machines/:id` | Get one machine  |
| POST   | `/machines`     | Create a machine |
| PATCH  | `/machines/:id` | Edit a machine   |
| DELETE | `/machines/:id` | Delete a machine |

👉 **View API / Check Readiness**: You can visit [https://pl-forecaster-portal.onrender.com](https://pl-forecaster-portal.onrender.com) to verify that the JSON Server is up and running.
